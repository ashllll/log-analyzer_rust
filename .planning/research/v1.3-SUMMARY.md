## 关键发现

### 1. 自定义过滤器 (Custom Filters)

**推荐模式:**
- 侧边栏快捷访问 + 模态对话框创建/编辑
- 与现有 FilterPalette 无缝集成
- 工作区级别隔离

**实现路径:**
1. 过滤器数据模型 + Rust FFI 接口
2. 保存/加载过滤器（后端持久化）
3. 侧边栏过滤器面板 UI
4. 过滤器创建/编辑对话框
5. 搜索栏集成与快捷操作

### 2. 多工作区标签页 (Multi-Workspace Tabs)

**推荐模式:**
- Tab bar with PageView/IndexedStack（VSCode/Chrome 风格）
- Riverpod family providers 隔离每个标签页状态
- 共享优先本地存储保存标签列表

**实现路径:**
1. Tab 基础设施 - 核心标签栏、开/关/切换
2. 工作区隔离 - Family providers、懒加载、资源清理
3. Tab 特性 - 拖拽排序、固定/取消、右键菜单、快捷键
4. 持久化与设置 - Tab 布局持久化、会话恢复

### 3. 日志级别统计 (Log Level Statistics)

**推荐模式:**
- 使用现有 fl_chart 包
- 主视图: 条形图（级别对比）
- 次视图: 饼图（分布）
- 趋势: 折线图（实时）

**实现路径:**
1. MVP: 统计卡片 (4 级别) + 饼图 + 5 秒自动刷新
2. 增强: 时间序列趋势图、触摸交互、历史对比
3. 高级: 实时流式更新、导出统计、告警阈值

### 4. 集成模式

**关键集成点:**
- Filters ↔ Tabs: 切换标签时保存/恢复过滤器状态
- Filters ↔ Search: 应用过滤器触发搜索更新
- Stats ↔ Tabs: 统计信息按工作区隔离

**状态管理:**
- 扩展现有 AppState 而非创建新 provider
- 使用 select() 优化细粒度重建
- 统计缓存 30 秒 TTL
