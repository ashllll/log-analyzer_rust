  <name>Task 2: 创建 RegexInputField 组件</name>
  <files>

  <action>
创建正则表达式输入框，支持实时语法验证。

1. **组件功能:**
   - 继承 StatefulWidget (需要管理 Timer 和验证状态)
   - 使用 FFI `validateRegex` API 进行验证 (不使用 Dart RegExp，保持与后端一致)
   - 300ms 防抖 (Timer) 避免频繁调用
   - 显示验证图标 (勾选/错误) 和 helper 文本

2. **状态管理:**
   - `_validationResult` 存储验证结果
   - `_debounceTimer` 防抖定时器
   - 在 `dispose()` 中取消 Timer (防止内存泄漏)

3. **UI 设计:**
   - TextField with InputDecoration
   - suffixIcon: 验证状态图标
   - errorText: 无效时显示错误信息
   - helperText: 有效时显示 "语法有效"

4. **关键代码模式:**
```dart
Future<void> _validateRegex() async {
  final bridge = BridgeService.instance;
  final result = await bridge.validateRegex(pattern);
  setState(() => _validationResult = result);
}
```

5. **避免:**
   - 不要在 build() 中调用异步方法
   - 不要硬编码颜色，使用 AppColors
   - 不要忘记 Timer 取消
