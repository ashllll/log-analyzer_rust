# Phase 2: 工作区与文件导入 - Context

**Gathered:** 2026-03-01
**Status:** Ready for planning

<domain>
## Phase Boundary

用户可以管理工作区（创建、打开、删除）、查看工作区状态，以及导入文件夹和压缩包（ZIP/TAR/GZ/RAR/7Z）到工作区。导入过程中显示进度，支持拖放和文件选择器两种方式。

</domain>

<decisions>
## Implementation Decisions

### 工作区管理 UI
- **布局**: 卡片布局，每个工作区作为卡片展示
- **信息展示**: 完整信息（名称、创建时间、文件数量、索引状态、总大小）
- **空状态**: 引导创建（显示引导图和「创建工作区」按钮）
- **删除确认**: 确认对话框
- **状态指示**: 状态指示器（就绪/索引中/错误，用颜色和图标区分）
- **名称**: 创建时自定义名称，可包含中文
- **排序**: 最近优先（最近打开的 3 个显示在最前）
- **打开方式**: 点击打开
- **位置**: 左右布局（左侧工作区列表，右侧内容）
- **编辑**: 点击设置图标弹出编辑面板
- **刷新**: 手动刷新按钮
- **数量限制**: 无限制
- **图标风格**: Material Icons
- **操作按钮**: 悬停显示
- **选中状态**: 边框+背景高亮
- **批量操作**: 不支持
- **搜索**: 支持按名称搜索
- **主题**: 自动适配深色/浅色主题
- **触摸设备**: 长按菜单
- **数据存储**: 本地存储
- **创建流程**: 表单创建（填写名称和选择文件夹）
- **文件选择**: 系统对话框
- **重复处理**: 提示用户重命名
- **打开体验**: 加载中提示
- **错误处理**: 友好提示+操作
- **权限错误**: 提示重新选择
- **迁移**: 不支持迁移
- **离线**: 完全离线可用
- **文件夹监控**: 创建时自动监控
- **大小显示**: 智能单位（KB/MB/GB）
- **首次打开**: 自动索引
- **索引位置**: 同目录
- **子文件夹**: 递归扫描
- **文件类型**: 仅索引日志文件（.log, .txt, .json）
- **隐藏文件**: 跳过以 . 开头的文件
- **临时文件**: 自动跳过（tmp, temp, cache）
- **备份**: 不提供备份功能
- **关闭行为**: 保存状态
- **键盘操作**: 支持键盘导航（上下键选择，回车打开）
- **快捷键**: 标准快捷键（Ctrl+N 新建, Ctrl+O 打开, Delete 删除）

### 文件导入流程
- **导入方式**: 同时支持拖放和按钮选择
- **多选**: 支持一次选择多个文件或多个文件夹
- **压缩包处理**: 自动检测并提示用户选择操作
- **重复文件**: 提示用户选择操作
- **拖放区域**: 指定区域（特定拖放区域）
- **拖放反馈**: 视觉反馈（高亮边框和提示文字）
- **文件大小限制**: 无限制
- **大文件**: 流式处理，显示进度
- **导入目标**: 当前工作区
- **文件过滤**: 仅允许日志相关文件（.log, .txt, .json 等）
- **导入确认**: 确认后导入
- **存储方式**: 复制到工作区
- **压缩包操作**: 解压后导入
- **压缩包预览**: 解压全部
- **错误处理**: 跳过失败文件继续导入，结束后报告
- **导入历史**: 保存历史
- **撤销**: 支持撤销
- **取消**: 导入进行中可取消
- **递归扫描**: 自动递归
- **符号链接**: 跳过
- **空文件夹**: 提示用户
- **权限错误**: 跳过继续
- **导入完成**: 显示完成通知
- **后台导入**: 支持后台导入

### 导入进度显示
- **展示方式**: 模态对话框
- **进度信息**: 详细信息（已处理文件数、总文件数、当前文件、预估时间）
- **单文件进度**: 大文件显示解压/处理进度
- **取消操作**: 取消按钮
- **进度条样式**: 圆形进度
- **错误显示**: 错误列表
- **完成显示**: 摘要报告
- **多任务**: 单任务
- **预估时间**: 显示预估剩余时间
- **速度显示**: 显示处理速度
- **已完成列表**: 显示已处理文件列表
- **更新频率**: 实时更新
- **暂停**: 支持暂停和继续
- **任务排序**: 按顺序
- **日志**: 保存详细导入日志

### 压缩包处理
- **格式支持**: 全部格式（ZIP, TAR, GZ, RAR, 7Z）
- **RAR 处理**: 使用 Rust 原生支持的方案
- **7Z 处理**: 使用 sevenz-rust 库
- **密码压缩**: 提示输入密码
- **损坏处理**: 报告损坏的文件，继续处理其他
- **解压位置**: 工作区目录下的临时文件夹
- **浏览支持**: 导入前可浏览压缩包内容，选择要导入的文件
- **大压缩包**: 流式解压
- **嵌套压缩**: 递归解压
- **删除原文件**: 不删除原压缩包
- **编码处理**: 自动检测，失败则使用 UTF-8
- **重新压缩**: 不覆盖，保持原压缩比
- **部分解压**: 支持仅解压选择的文件

### Claude's Discretion

- 错误状态处理
- 主题细节（颜色、阴影）
- 具体组件实现细节

</decisions>
