### 修复 2: `test_standard_engine` 测试用例

**文件**: `src-tauri/src/services/regex_engine.rs`
**修改内容**:

* 将测试模式从 `(?<=foo)\d+` (lookbehind) 改为 `\d+` (普通正则)

* 这样可以正确测试 StandardEngine 的基本功能，而不需要 lookbehind 支持

### 修复后的测试用例

```rust
#[test]
fn test_lookaround_detection() {
    assert!(needs_lookaround(r"(?<=foo)bar"));  // lookbehind
    assert!(needs_lookaround(r"bar(?=foo)"));   // lookahead
    assert!(needs_lookaround(r"(?!test)"));     // negative lookahead
    assert!(!needs_lookaround(r"\d+"));         // 无 lookaround
    assert!(!needs_lookaround("error"));        // 无 lookaround
}

#[test]
fn test_standard_engine() {
    let engine = StandardEngine::new(r"\d+").unwrap();  // 使用普通正则
    let text = "foo123 and foo456";
    let matches: Vec<_> = engine.find_iter(text).collect();
    assert_eq!(matches.len(), 2);
}
```
